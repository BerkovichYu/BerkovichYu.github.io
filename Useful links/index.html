Suggestions to graduates

USTC Course Schedule Guide. This is a set of course learning suggestions voluntarily compiled by graduates from the undergraduate mathematics program at USTC. Written in Chinese, it remains valuable to this day, even though the students' perspectives are somewhat limited.
